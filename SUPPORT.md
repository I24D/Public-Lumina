# Support

Use this repository for public installation and release questions about Lumina OpenClaw Desktop.

## Before Opening an Issue

Please include:

- Operating system and version.
- Lumina OpenClaw version.
- Installer filename used.
- What you expected to happen.
- What actually happened.
- Screenshots or logs when available.

## Do Not Share Secrets

Do not post API keys, passwords, tokens, private `.env` files, Supabase credentials, Render credentials, or GitHub tokens in public issues.

## Current Supported Public Version

Only `lumina-desktop-v1.0.10` is currently supported in this public repository.
