# Release Assets

This repository publishes official Lumina OpenClaw Desktop release assets.

## User Installer

`Lumina.OpenClaw_1.0.10_x64-setup.exe`

Use this file to install Lumina OpenClaw Desktop on Windows x64.

## Runtime Assets

`lumina-openclaw-runtime-bundle-1.0.10-win32-x64.tar.gz`

Contains the bundled OpenClaw runtime used by Lumina Desktop. This is not the normal user installer.

`lumina-openclaw-runtime-bundle-1.0.10-win32-x64.manifest.json`

Describes runtime bundle contents and verification metadata.

## Release Manifest

`lumina-release-1.0.10-win32-x64.manifest.json`

Top-level release metadata used for verification and future update automation.

## Bootstrapper

`lumina-bootstrapper-1.0.10-win32-x64.tar.gz`

Bootstrapper artifact used by Lumina runtime/update tooling.

## Policy

Only current, supported public release assets should remain published here. Old or experimental assets should be removed or moved to an internal repository to keep the public distribution clean.
